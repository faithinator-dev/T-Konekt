2Click "Users" in sidebar ‚Üí "Add New User" button

üì¢ How to Send Broadcast Message

1Go to "Broadcast Messages" in admin sidebar

2Select recipients: All Users / Active Users / Groups

3Choose message type: Notification / Message / Alert

4Enter title and message content

5Click "Send Now" ‚Üí Users receive notification!

Tip: Users can see notifications by clicking the bell üîî icon

üìß How to Send Mass Email
