# T-Konekt Logo

## Logo File
Place your logo file as: `logo.png` in this directory

## Current Usage
The T-Konekt logo is used in:
- **Login page**: Large logo (120px height) next to brand name
- **All navigation bars**: Small logo (40px height) in navbar
  - Home
  - Profile
  - Friends
  - Messages
  - Groups
  - Watch
  - Notifications

## Specifications
- **Format**: PNG with transparent background
- **Recommended size**: 512x512px or higher
- **Colors**: Blue gradient (#0891b2 to #0e7490)
- **Style**: Modern, clean, and recognizable

## How to Update
Simply replace `logo.png` with your new logo file. The logo will automatically appear across all pages.
